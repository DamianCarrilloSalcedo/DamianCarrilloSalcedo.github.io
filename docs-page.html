Empaquetado Last updated: 2019-06-01

Para empaquetar la aplicacion primero tenemos que tener instalado java. Para instalar java tienes que Pulsar aqui Una vez que lo tenemos instalado, tendremos que meterlo en las variables del sistema (path) pulsando en busqueda y editar las variables del sistema.

Cuando se nos abra, tenemos que pulsar en editar variables del sistema y posteriormente en nueva. Tendra esta informacion.

Ya tenemos nuestro java configurado y podemos proceder al empaquetado.

Proceso de Empaquetado

Para empaquetar tenemos dos formas, mediante consola o mediante el Ide(Entorno de desarrollo). En mi caso voy a explicar mediante el Ide. Yo estoy usando netbeans y para cada ide es un proceso diferente. Para generar el jar en netbeans debemos de introducir este codigo dentro de nuestro pom.
Una vez tengamos ese codigo en el pom, en netbeans debemos de pulsar el icono del martillo y se nos generara el jar

Ya se ha generado nuestro jar en nuestro proyecto, en la carpeta de target como podemos ver en la imagen

Proceso de Ejecucion

Ejecucion Empaquetado

Para ejecutar el empaquetado tenemos que hacer click derecho y pulsar abrir el terminal

Una vez que estamos en la consola usamos el codigo java -jar CasaDelTe-1.0-SNAPSHOT.jar

Como podemos ver se ha ejecutado correctamente el jar

Uso de la aplicacion

Estamos en la pagina principal donde tendremos varias opciones. Primero login donde podremos dar nuestro usuario y clave para entrar en la tienda a comprar, registro para introducir nuestros datos y posteriormente hacer el login, salir para cerrar la aplicacion y ayuda para mostrar la ayuda de nuestra aplicacion

Registro

Pulsamos en registro e introducimos todos los campos. Finalmente usamos el boton de registrar para introducir los datos

Pulsamos en volver para hacer el login del usuario creado.

Una vez de vuelta en la pantalla principal pulsamos en el boton login.

Login

Esta es nuestra ventana Login

Metemos los datos de nuestro usuario creado anteriormente y pulsamos en login. Si todo va bien te enviara a la pagina de productos

Una vez en la pagina productos, podemos desloguearnos pulsando el boton cerrar sesion que nos llevara a la pagina principal

Ayuda

Al pulsar el boton nos llevara a nuestra pagina de ayuda
